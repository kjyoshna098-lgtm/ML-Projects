ML Projects - Sales Data Analysis

Project Overview
This project uses Machine Learning techniques to analyze Amazon sales data and help optimize discounts and pricing strategies.
The goal is to improve sales performance and reduce unnecessary discounts using advanced ML models.
Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib

 Implemented Models
## 1. Clustering
Customer segmentation using clustering techniques to group similar buyers.
File: clustering.py

### 2. XGBoost Model
Predictive model to analyze sales patterns and recommend optimized discount strategies.
File: xgboost.py

### 3. Libraries
Contains required library imports for the project.
File: libraries.py
